# fikalist

[![DOI](https://zenodo.org/badge/130894835.svg)](https://zenodo.org/badge/latestdoi/130894835)  

Automatic fikalist generator. You may cite the above DOI for any disputes.
See ```fikalist.ipynb``` for how to use.

* numpy

* holidays
